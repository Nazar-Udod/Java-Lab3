Інструкція до використання:
- Переконайтеся, що Java встановлена на вашій системі
- Відкрийте термінал
- Скопіюйте репозиторій на свою систему за допомогою команди git clone або завантажте його іншим чином
- Перейдіть у необхідний каталог за допомогою команди cd [шлях до репозиторію]\src
- Скомпілюйте програму за допомогою команди javac Main.java
- Запустіть програму за допомогою команди java Main
- Введіть кількість літаків, які хочете додати до масиву
- За підказками у консолі введіть характеристики усіх літаків
- Програма виведе масив усіх літаків у порядку введення
- Програма відсортує масив за зростанням швидкості, при однаковості - за спаданням року створення
- Введіть характеристики літака, який хочете знайти у масиві
- Якщо літак з ідентичними значеннями полів є, буде виведено його номер у відсортованому масиві (нумерація починається з одиниці)
- В іншому випадку, виведеться повідомлення про відсутність ідентичного літака
